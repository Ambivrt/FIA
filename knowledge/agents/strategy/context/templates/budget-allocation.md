# Mall: Budgetfördelning

## Format

- **Syfte:** Optimera marknadsbudgeten över kanaler och aktiviteter
- **Godkännande:** Kräver Orchestrator-godkännande (eskalering vid > 50 000 SEK)

## Struktur

### 1. Total budget

- Total tillgänglig budget (period)
- Jämförelse med föregående period
- Eventuella begränsningar eller öronmärkningar

### 2. Fördelning per kanal

| Kanal                 | Budget (SEK) | Andel (%) | Historisk ROI | Föreslagen justering |
| --------------------- | ------------ | --------- | ------------- | -------------------- |
| SoMe (organiskt)      |              |           |               |                      |
| Annons (LinkedIn Ads) |              |           |               |                      |
| Annons (Google Ads)   |              |           |               |                      |
| Webb/SEO              |              |           |               |                      |
| Event                 |              |           |               |                      |
| Content (produktion)  |              |           |               |                      |
| Övrigt                |              |           |               |                      |

### 3. Historisk ROI-analys

- ROI per kanal (senaste 3 kvartal)
- CPL per kanal
- Konverteringsgrad per kanal
- Pipeline-bidrag per kanal i SEK

### 4. Benchmark-jämförelse

- Intern historik (YoY)
- Branschbenchmark (B2B tech/hållbarhet)
- Kanalspecifika benchmarks (CPC, CPL, engagement)

### 5. Scenarioanalys

#### Konservativ (-20%)

- Fördelning, förväntade resultat, risker

#### Neutral (föreslagen)

- Fördelning, förväntade resultat, risker

#### Aggressiv (+30%)

- Fördelning, förväntade resultat, risker

### 6. Rekommendation

- Rekommenderat scenario (motiverat)
- Viktigaste omfördelningar vs föregående period
- Förväntad pipeline-impact

### 7. KPI:er

- Total pipeline-bidrag (SEK)
- Blended CPL
- Blended CAC
- Marketing-sourced revenue (%)
- Budget efficiency ratio

## Riktlinjer

- Basera på faktisk data, inte branschgenomsnitt
- Inkludera alltid tre scenarion
- Budget > 50 000 SEK eskaleras automatiskt
- Omfördela budget månadsvis baserat på Analytics Agent-data
