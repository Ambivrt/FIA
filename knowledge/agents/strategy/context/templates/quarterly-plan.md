# Mall: Kvartalsplan

## Format
- **Tidsperiod:** Q1–Q4
- **Godkännande:** Kräver Orchestrator-godkännande

## Struktur

### 1. Kvartalsmål
- 2–3 övergripande marknadsföringsmål
- Kopplade till Forefronts affärsmål
- SMART-formulerade (Specifika, Mätbara, Uppnåeliga, Relevanta, Tidsbundna)

### 2. Fokusområden
- Vilka kanaler och aktiviteter som prioriteras
- Motivering baserad på data från föregående kvartal
- Resurstilldelning per fokusområde

### 3. Budget
- Total budget i SEK
- Fördelning per kanal/aktivitet
- Jämförelse med föregående kvartal

### 4. KPI:er
- 4–6 nyckeltal med målvärden
- Baseline från nuläge
- Uppföljningsfrekvens (vecka/månad)

### 5. Milstolpar
- Veckovis tidslinje med leveranser
- Beroenden mellan aktiviteter
- Ansvarsfördelning (vilken agent ansvarar)

### 6. Risker
- Identifierade risker och mitigering
- Beredskapsplan vid underprestation

## Datakällor
- GA4 (föregående kvartals trafik och konverteringar)
- HubSpot (lead pipeline, konverteringsgrad)
- Innehållsarkiv (vad som fungerat/inte fungerat)
