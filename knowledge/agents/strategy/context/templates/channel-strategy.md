# Mall: Kanalstrategi

## Format

- **Syfte:** Strategi för en specifik marknadskanal
- **Godkännande:** Kräver Orchestrator-godkännande
- **Kanal:** Bestäms av `channel`-fältet i content_json (some, web, event, ads, earned)

## Struktur

### 1. Kanal och nuläge

- Vilken kanal gäller strategin?
- Nuvarande prestanda (data från GA4/HubSpot/plattform)
- Befintliga aktiviteter och resultat

### 2. Mål

- Kanalspecifika mål (kopplade till kvartalsplan)
- SMART-formulerade målvärden
- Tidhorisont

### 3. Målgrupp

- Primär målgrupp på denna kanal
- Persona-koppling
- Beteendemönster på kanalen

### 4. Taktik

- Specifika aktiviteter och format
- Publiceringsfrekvens/närvaro
- Content-typer som fungerar på kanalen
- Tonalitet anpassad till kanalen

### 5. Budget

- Kanalbudget i SEK
- Fördelning (organiskt vs betalt, om relevant)
- Förväntad ROI baserat på historik

### 6. KPI:er

- 4-6 kanalspecifika KPI:er med målvärden
- Mätmetod och verktyg
- Rapporteringsfrekvens

### 7. Tidplan

- Implementeringsmilstolpar
- Utvärderingspunkter
- Koppling till övrig planering

## Riktlinjer

- Strategin ska vara tillräckligt specifik för att Content/Campaign Agent kan exekvera
- Basera alltid på data — inkludera benchmark-jämförelser
- Koppla till pågående kvartalsplan
