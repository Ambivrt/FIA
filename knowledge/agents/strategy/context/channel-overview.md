# Forefronts kanalöversikt

## Aktiva kanaler

### Primära (always-on)

1. **LinkedIn** — Thought leadership, employer branding, lead gen
2. **Webbplats** — Content hub, konvertering, SEO
3. **E-post** — Nurture, newsletter, kampanjer

### Sekundära (kampanjdrivna)

4. **LinkedIn Ads** — Betald räckvidd, lead gen forms, retargeting
5. **Google Ads** — Search (intent-driven), Display (retargeting)
6. **YouTube** — Webinar-inspelningar, tutorials, case studies

### Periodiska

7. **Event** — Mässor, egna webinar, talarmöjligheter
8. **Earned/PR** — Gästinlägg, pressmeddelanden, expertkommentarer

## Kanalsynergier

```
LinkedIn-inlägg → Bloggpost → Newsletter → LinkedIn Ads (boost)
Webinar → YouTube → LinkedIn clips → Blogg-sammanfattning
Case study → LinkedIn → Google Ads landing page → Nurture e-post
```

## Regler för kanalval

1. Välj kanal baserat på målgruppens beteende (var finns de?)
2. Max 2-3 primära kanaler per kampanj
3. Återanvänd content cross-channel — producera en gång, distribuera brett
4. Betald media ska komplettera organiskt — inte ersätta
5. Mät alltid attribution per kanal
