# Mall: Kvartalsöversikt

## Format
- **Leverans:** Sista fredagen i kvartalet
- **Längd:** 1 000–1 500 ord
- **Ton:** Strategisk, datadriven, framåtblickande

## Struktur

### 1. Executive Summary (200 ord)
- Kvartalets viktigaste resultat
- Mål vs. utfall (övergripande)
- Strategiska rekommendationer för nästa kvartal

### 2. Måluppfyllelse
| Mål | Målvärde | Utfall | Status |
|-----|----------|--------|--------|
| | | | Uppnått / Ej uppnått / Överträffat |

### 3. Trafikanalys
- Trender över kvartalet (vecka för vecka)
- Kanalfördelning (organisk, direkt, referral, betald)
- Jämförelse med föregående kvartal och samma kvartal förra året

### 4. Innehållsprestanda
- Totalt producerat innehåll (typ och antal)
- Top 5 mest besökta sidor
- Innehåll med högst konverteringsgrad
- Brand Agent: godkännandegrad och vanligaste feedback

### 5. Lead & Pipeline
- Leads genererade per kanal
- MQL-konverteringsgrad
- Pipeline-bidrag i SEK
- Jämförelse med föregående kvartal

### 6. Kostnadsanalys
| Kategori | Budget | Faktiskt | Avvikelse |
|----------|--------|----------|-----------|
| LLM-kostnader | | | |
| Annonsering | | | |
| Verktyg/integrationer | | | |
| **Totalt** | | | |

### 7. Trender & Omvärld
- 2–3 marknadstrender som påverkar nästa kvartal
- Konkurrentaktivitet
- Möjligheter att utnyttja

### 8. Rekommendationer för nästa kvartal
- 3–5 prioriterade åtgärder
- Resursfördelning
- Föreslagna KPI-justeringar

## Riktlinjer
- Balanser data med insikter – siffror utan analys har inget värde
- Var ärlig om missade mål – analysera varför
- Avsluta med en tydlig riktning framåt
