# Mall: Bildgenerering

## Format

- **Syfte:** Generera bilder via Nano Banana 2 (Gemini Flash Image)
- **Modell:** nano-banana-2
- **Godkännande:** Brand Agent granskar visuellt (multimodal)

## Promptriktlinjer

### Stil och varumärke

- Forefronts visuella identitet: jordnära, professionell, human
- Färgpalett: dova toner (#7D5365 plum, #42504E forest, #555977 slate, #756256 stone, #7E7C83 ash)
- Gradient-accent tillåten: #FF6B0B → #FFB7F8 → #79F2FB
- Bildstil: clean, modern, skandinavisk estetik
- Undvik: stockfoto-känsla, generiska kontorsbilder, överbearbetade filter

### Promptstruktur

1. **Motiv:** Vad ska bilden visa? Var konkret.
2. **Stil:** Fotorealistisk, illustration, abstrakt, grafisk?
3. **Stämning:** Professionell, varm, innovativ, framtidsorienterad?
4. **Komposition:** Utrymme för text? Porträtt/landskap? Fokusområde?
5. **Varumärkeskoppling:** Hur knyter bilden an till Forefronts identitet?

### Förbjudna element

- Generiska "handshake"-bilder
- Stockfoto-mönster (perfekta leenden, sterila kontor)
- Överdrivna AI/tech-visuella (neon, hologram, robotar)
- Text i bilden (hanteras separat)
- Forefronts logotyp i bilden (läggs till i post-produktion)

## Kvalitetskontroll

Brand Agent granskar varje genererad bild mot:

1. Färgpalett-överensstämmelse
2. Varumärkeskänsla och ton
3. Professionell kvalitet
4. Relevans för innehållet
