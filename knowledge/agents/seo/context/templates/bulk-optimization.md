# Mall: Bulk SEO-optimering

## Format

- **Syfte:** Optimera flera befintliga sidor/artiklar i en batch
- **Modell:** Claude Sonnet (kostnadsoptimerad för bulk)
- **Godkännande:** 5% sample review

## Struktur

### 1. Optimeringsöversikt

- Antal sidor som analyseras
- Gemensamma problem identifierade
- Förväntad påverkan (ranking-förbättring, trafik)

### 2. Per sida

#### URL: [sidans URL]

- **Nuvarande ranking:** Position för primärt keyword (om känd)
- **Primärt keyword:** Sökterm sidan bör ranka för
- **Titel-optimering:** Nuvarande → Föreslagen (max 60 tecken)
- **Meta-description:** Nuvarande → Föreslagen (max 155 tecken)
- **H1-optimering:** Justering om nödvändig
- **Innehållsluckor:** Vad saknas jämfört med top-3 rankade sidor?
- **Intern länkning:** Föreslagna interna länkar (från/till)
- **Schema markup:** Rekommenderad strukturerad data

### 3. Prioritering

Sidor sorterade efter:

1. Största potentiella trafikvinst
2. Lägsta insats (quick wins först)
3. Strategisk betydelse

## Riktlinjer

- Ändra aldrig sidans huvudbudskap — optimera kring det
- Bevara Forefronts ton i alla textförslag
- Fokusera på sidor som rankar position 5-20 (störst potential)
- Undvik kannibalisering — en sida per primärt keyword
- Inkludera alltid before/after för varje förändring
