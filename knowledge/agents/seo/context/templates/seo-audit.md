# Mall: SEO-audit

## Format
- **Typ:** Teknisk + innehållsmässig SEO-analys
- **Frekvens:** Månatlig
- **Output:** Strukturerad rapport med prioriterade åtgärder

## Struktur

### 1. Sammanfattning
- Övergripande SEO-hälsa (poäng eller betyg)
- Top 3 prioriterade åtgärder
- Jämförelse med föregående period

### 2. Teknisk SEO
- Crawlbarhet och indexering
- Sidladdningstid (Core Web Vitals)
- Mobilanpassning
- Strukturerad data (schema.org)
- Brutna länkar och omdirigeringar

### 3. On-page SEO
- Title tags och meta descriptions (saknade, för långa, duplicerade)
- Rubrikhierarki (H1, H2, H3)
- Intern länkning
- Bildoptimering (alt-texter, filstorlek)

### 4. Innehållsanalys
- Content gaps – ämnen konkurrenter rankar för men inte vi
- Thin content – sidor med för lite innehåll
- Kannibalisering – sidor som konkurrerar om samma nyckelord
- Möjligheter för uppdatering av befintligt innehåll

### 5. Keyword-ranking
- Top 20 nyckelord med position
- Positionsförändringar sedan förra auditen
- Nya möjligheter (position 11–30)

### 6. Prioriterade åtgärder
- Numrerad lista, sorterad efter förväntad påverkan
- Varje åtgärd: vad, varför, estimerad effekt

## Riktlinjer
- Fokusera på svenskspråkiga söktermer
- Prioritera long-tail med kommersiell avsikt
- Alla rekommendationer ska vara konkreta och exekverbara
