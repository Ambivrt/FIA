# Mall: Keyword Research

## Format

- **Syfte:** Identifiera och prioritera söktermer för Forefronts innehållsstrategi
- **Modell:** Google Search (primär datakälla) + Claude Opus (analys)
- **Triggad av:** Strategy Agent (channel_strategy) eller Intelligence Agent (trend insights)
- **Godkännande:** 5% sample review

## Struktur

### 1. Sökordsöversikt

- Totalt antal identifierade termer
- Gruppering per tema/kluster
- Språkfokus: Svenska (primärt), Engelska (om internationell relevans)

### 2. Keyword-kluster (3-5 kluster)

Per kluster:

- **Tema:** Beskrivande namn
- **Primärt keyword:** Högst prioriterat sökord
- **Sekundära keywords:** 5-10 relaterade termer
- **Long-tail varianter:** 3-5 specifika fraser med kommersiell intent
- **Sökvolym:** Uppskattad (hög/medium/låg) baserat på sökresultat
- **Konkurrens:** Hög/Medium/Låg baserat på SERP-analys
- **Content-gap:** Har Forefront content för detta? Ja/Delvis/Nej
- **Rekommenderad content-typ:** Blog post, landing page, case study, whitepaper

### 3. Prioriteringsmatris

| Keyword | Sökvolym | Konkurrens | Relevans | Prioritet |
| ------- | -------- | ---------- | -------- | --------- |
| ...     | H/M/L    | H/M/L      | 1-10     | P1/P2/P3  |

### 4. Handlingsplan

- **P1 (0-4 veckor):** Quick wins — låg konkurrens, hög relevans
- **P2 (1-3 månader):** Strategiska keywords — medium konkurrens
- **P3 (nästa kvartal):** Long-term plays — hög konkurrens, hög volym

## Riktlinjer

- Fokusera på B2B-termer relevanta för Forefronts tjänster
- Prioritera kommersiell och transaktionell intent framför informationell
- Inkludera lokala svenska termer ("AI-konsulting Stockholm" etc.)
- Analysera top-3 konkurrenters ranking för varje kluster
- Koppla varje keyword till ett specifikt content-format
