# SEO Geo-riktlinjer

## Primärt fokus: Sverige

### Språk

- Allt innehåll på svenska som standard
- Engelska enbart för internationella sidor eller branschtermer utan bra svensk översättning
- Undvik svengelska – använd svenska termer där de finns

### Sökbeteende

- Google.se dominerar (~95% marknadsandel)
- Svenskar söker ofta med frågeformat ("hur gör man...", "vad kostar...")
- Long-tail-sökningar vanliga inom B2B-tech

### Lokala söktermer

Prioritera termer med kommersiell avsikt:

- "[tjänst] + konsult/partner/leverantör"
- "[bransch] + digitalisering/automatisering"
- "[problem] + lösning/hjälp"
- Städer: Stockholm, Göteborg, Malmö (vid lokalt fokus)

### Tekniskt

- hreflang: `sv-SE` som primärt
- Domän: .se (forefront.se)
- Server: EU-baserad (GDPR)

## Konkurrenslandskap

- Analysera svenska tech-konsultbolags SEO-strategi
- Identifiera content gaps i svensk B2B-tech
- Bevaka trender via Google Trends (region: Sverige)

## Riktlinjer

- Undvik att optimera för termer vi inte kan leverera på
- Kvalitet framför volym i keyword-strategi
- Uppdatera keyword-rankings.json månadsvis
